Теорія щодо тегу таблиць

Основні теги таблиці:

  • <table> </table> - контейнер для всієї таблиці
  • <tr> </tr> - рядок таблиці.
  • <td> </td> - омірка (дані).
  • <th> </th> - комірка заголовку.
  • <caption> </caption> - заголовок таблиці.
  • <thead>, <tbody>, <tfoot> — теги групування рядків.

Основними атрибутами тегу table є:

  • border — задає товщину рамки таблиці (проте краще через CSS).
  • cellpadding — внутрішній відступ у комірках (проте краще через CSS).
  • cellspacing — відстань між комірками (проте краще через CSS `border-spacing`)
  • width — ширина таблиці (можна в пікселях або %).
  • height — висота таблиці (рідко використовується, краще через CSS).
  • align — вирівнювання таблиці (застарілий, краще через CSS).

Основними атрибутами тегу tr є:

  • align — вирівнювання вмісту рядка ("left", "center", "right").
  • valign — вертикальне вирівнювання ("top", "middle", "bottom").
  • bgcolor — колір фону (проте краще через CSS).

Основними атрибутами тегів td і th є:

  • colspan — об’єднання комірок по горизонталі (кількість колонок).
  • rowspan — об’єднання комірок по вертикалі (кількість рядків).
  • headers — зв’язує комірку з заголовком (для доступності).
  • scope — визначає область заголовку: "row", "col", "rowgroup", "colgroup".
  • abbr — коротка форма заголовку для доступності.
