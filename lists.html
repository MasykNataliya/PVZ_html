Основна інформація

Приклад коду списків:



       <div class="hints">
    <h2>Як користуватися сайтом</h2>
    <ol>
      <li> Переглянь список завдань у таблиці «Графік завдань».</li>
      <li> Заповни форму «Додати нове завдання» та натисни кнопку <strong>Додати</strong>.</li>
      <li> Вибери дату у полі <strong>Дата</strong>.</li>
      <li> Обери статус: «Заплановано», «В процесі» або «Виконано».</li>
      <li> Натисни <strong>Видалити</strong> для видалення завдання.</li>
    </ol>
  </div>

Теорія щодо тегу списків

Список додається до веб сторінки за допомогою тегів:

  1. <ol> </ol> - нумерований список.

    2. Основними атрибутами цього тегу є:

    • type — визначає стиль нумерації, наприклад "1" - арабські цифри, "А" - великі латинські літери, "І" - великі римські цифри, "і" т.д.
    • start — задає початкове число для нумерації.
    • reversed — відображає список у зворотному порядку.

  2. <ul> </ul> - маркований список.

    3. Основними атрибутами цього тегу є:

    • type — визначає вид маркерів: disc → чорна крапка (за замовчуванням), circle → порожнє коло, square → квадрат

  3. <li> </li> - елемент списку

    4. Основними атрибутами цього тегу є:

    • value — задає конкретне значення нумерації для елемента у нумерованому списку
