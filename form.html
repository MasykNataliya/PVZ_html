Теорія щодо тегу форми

Форма додається до веб сторінки за допомогою тегу <form> </form>

Основними атрибутами тегу є:

  • action - Вказує адресу (URL), куди надсилаються дані форми після відправлення.
  • method - Визначає спосіб передачі даних: get (у рядку URL) або post (у тілі запиту).
  • enctype - Формат кодування даних при відправці. Використовується з post.
  • target - Визначає, де відкриється результат: self (за замовчуванням, у тому ж вікні) blank (нове вікно/вкладка)
  • autocomplete - Керує автозаповненням полів: on або off.
  • novalidate - Вимикає вбудовану перевірку валідності браузером.
  • name - Ім’я форми, яке можна використовувати у JavaScript.
  • accept-charset - Визначає набір символів для відправки даних (наприклад, "UTF-8")

Для внесення даних використовується тег input. Основні атрибути тегу є:

  • type — визначає тип поля ("text", "password", "email", "number", "date", "file", "checkbox", "radio", "submit").
  • name — ім’я поля, яке використовується при відправці даних.
  • value — значення поля (наприклад, текст у кнопці).
  • placeholder — текст‑підказка всередині поля.
  • required — робить поле обов’язковим для заповнення.
  • readonly — поле доступне лише для читання.
  • disabled — поле вимкнене.
  • maxlength — максимальна кількість символів.
  • min, max, step — обмеження для числових/датових полів.

Назву певного поля форми забезпечує тег label. Основними атрибутами тегу є:

  • for — зв’язує мітку з конкретним полем.

Також у формі часто використовується тег button. Основними атрибутами тегу є:

  • type — визначає поведінку кнопки: "submit" - відправляє форму на сервер (за замовчуванням, якщо атрибут не вказаний). "reset" - очищає всі поля форми, повертаючи їх до початкових значень. "button" - звичайна кнопка, яка сама по собі нічого не робить, але корисна для JavaScript‑обробників.
  • name — ім’я кнопки, яке може бути відправлене разом із даними форми.
  • value — значення кнопки, яке теж може бути передане при відправці.
  • disabled — робить кнопку неактивною.
  • autofocus — автоматично фокусується на кнопці при завантаженні сторінки.
  • form — дозволяє зв’язати кнопку з конкретною формою за її id, навіть якщо кнопка знаходиться поза формою.
  • formaction — перевизначає атрибут action форми.
  • formmethod — перевизначає метод відправки ("get" або "post") для конкретної кнопки.
